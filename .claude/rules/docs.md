# ドキュメント作成ルール

すべてのドキュメントは **日本語** で記述する。

## ディレクトリ規約

| ディレクトリ | 内容 | 担当 |
|-------------|------|------|
| `docs/strategy/` | 市場分析、競合インテリジェンス | CSO |
| `docs/product/` | PRD、ユーザーストーリー、仕様書 | CPO |
| `docs/marketing/` | コピー、ブランディング、ローンチ計画 | CMO |
| `docs/finance/` | コスト見積もり、価格モデル | CFO |

## 基本方針

- 仕様書、設計書、READMEは必ず日本語で記述する
- 技術的なコードコメントやコミットメッセージは英語で可
- サブエージェントの成果物（ドキュメント）も日本語必須
