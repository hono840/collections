# worldcup-kickoff 受入条件（Acceptance Criteria）

> 対象: MVP（P0 機能） / フォーマット: Given（前提）/ When（操作）/ Then（期待結果）
> QA エンジニアがそのままテストケースに落とせる粒度で記述する。
> 大会ファクトは PRD「0. 大会ファクト」を唯一の正とする（48 チーム / 12 グループ×4 / 104 試合 / 16 会場 / 6-11 開幕・7-19 決勝 / 上位2+3位上位8=32 進出 / 日本出場 / 新規初出場 4 国）。

---

## 共通の前提・用語
- **JST 表記**: 日時はすべて日本時間（UTC+9）で表示し、コピーに「（日本時間）」を明記する。
- **TBD 表記**: 抽選未確定の対戦相手は記号や空欄ではなく、初心者に分かる日本語（例:「対戦相手は抽選で決定」）で表示する。
- **推し国**: ユーザーが保存した応援国。localStorage キー `wck_favorite_country` に国コードを保持。
- **<Term>**: 本文中の専門用語に付くポップオーバートリガ。タップで用語じてんの語義を表示。

## F1. 開幕カウントダウン付きホーム + オンボーディング導線

### AC-F1-01 カウントダウン表示（開幕前）
- [ ] Given 現在日時が 2026-06-11 開幕より前、When ユーザーがホームを開く、Then 開幕までの残り日数が大きく表示され、開幕日「2026年6月11日 開幕（日本時間）」が併記される。

### AC-F1-02 開幕後の表示切替
- [ ] Given 現在日時が 2026-06-11 以降かつ 2026-07-19 以前、When ホームを開く、Then カウントダウンが「大会○日目」等の開催中表示に切り替わる。
- [ ] Given 現在日時が 2026-07-19 の決勝後、When ホームを開く、Then 「閉幕」相当の表示になり、残り日数のマイナス表示やゼロ表示の不整合が起きない。

### AC-F1-03 「3 分で W杯を知ろう」導線
- [ ] Given ホームを開いた初回訪問者、When 画面を見る、Then 「3 分で W杯を知ろう」CTA がファーストビュー内に視認でき、タップすると推し国診断（/quiz）またはオンボーディングに遷移する。

### AC-F1-04 今日の試合カード（ホーム）
- [ ] Given 当日に試合が存在する、When ホームを開く、Then 「今日の試合」が最大 3 件、対戦国・キックオフ時刻（日本時間）・会場とともに表示される。
- [ ] Given 当日に試合がない（休養日・開幕前）、When ホームを開く、Then 空表示ではなく「今日は試合がありません / 次の試合は○月○日」等の代替メッセージが出る（エッジ AC-E-01 参照）。

### AC-F1-05 推し国カード（ホーム）
- [ ] Given 推し国が未設定、When ホームを開く、Then 「推し国を見つけよう」診断 CTA が表示される。
- [ ] Given 推し国が設定済み、When ホームを開く、Then その国カード（国名・国旗・代表カラー）と、その国の次の試合（あれば日本時間）が表示される。

### AC-F1-06 主要導線
- [ ] Given ホーム、When 画面下部までスクロール、Then 「W杯のしくみを知る（ブラケット）」「用語を調べる（用語じてん）」への導線が存在し、それぞれ正しいページに遷移する。

### AC-F1-07 グローバルナビ
- [ ] Given 任意の P0 ページ、When 下部タブを見る、Then ホーム / 試合 / ブラケット / 国図鑑 / じてん の 4〜5 タブが常時表示され、現在地がハイライトされる。

## F2. 試合日程 / 今日の試合

### AC-F2-01 日付別一覧
- [ ] Given /matches を開く、When 一覧を見る、Then 試合が日付順（昇順）に並び、各試合に 対戦国・キックオフ時刻（日本時間）・会場・グループ名 が表示される。
- [ ] Given 全 104 試合のデータ、When 一覧をスクロール、Then データ欠落や重複表示なくすべての試合に到達できる。

### AC-F2-02 今日タブ
- [ ] Given /matches に「今日」タブがある、When 今日タブを選ぶ、Then 当日（日本時間基準）の試合のみが表示される。
- [ ] Given 当日に試合がない、When 今日タブを選ぶ、Then 「今日は試合がありません」と次の試合日への導線が表示される。

### AC-F2-03 試合詳細
- [ ] Given 一覧の試合をタップ、When /matches/[id] を開く、Then 対戦国・キックオフ時刻（日本時間）・会場（都市/スタジアム）・グループまたはラウンド名 が表示される。
- [ ] Given 試合詳細、When 対戦国名をタップ、Then 該当国の国ページ（/countries/[code]）に遷移する。

### AC-F2-04 ノックアウトの未確定カード
- [ ] Given 対戦カードが未確定なノックアウト試合（例: ラウンド32）、When 試合詳細を開く、Then 対戦国欄が TBD 表記（例:「グループ○○ 1位 vs ○○」または「抽選/結果で決定」）になり、空欄や「undefined」が出ない。

### AC-F2-05 時刻変換の正しさ
- [ ] Given UTC で保持された試合データ、When 一覧・詳細を表示、Then 表示時刻が日本時間（UTC+9）に正しく変換され、日付跨ぎ（深夜・早朝キックオフ）も正しい日付に分類される。

## F3. ブラケット / トーナメント表（グループ表含む）

### AC-F3-01 グループ表（12 組）
- [ ] Given /bracket を開く、When グループ表を見る、Then 12 グループ × 各 4 チーム が表示される。
- [ ] Given 各グループ、When 表を見る、Then 各国名・国旗が表示され、順位列（または順位の概念）が初心者に分かる形で示される。

### AC-F3-02 突破ルールの説明
- [ ] Given グループ表セクション、When 説明を読む、Then 「各組 上位 2 チーム + 3 位のうち成績上位 8 チーム = 計 32 チームが勝ち上がる」旨が初心者向けの言葉で明示される。
- [ ] Given 当該説明文に専門用語（例: グループステージ、ラウンド32）、When 用語をタップ、Then `<Term>` ポップオーバーで語義が出る。

### AC-F3-03 決勝トーナメント・ブラケット
- [ ] Given /bracket、When ブラケット図を見る、Then ラウンド32 → ラウンド16 → 準々決勝 → 準決勝 → 決勝 の段階が視覚的に示される。
- [ ] Given 対戦が未確定の段階、When ブラケットを見る、Then 該当枠が TBD 表記（例:「勝者が進出」）で表示され、空枠や記号のみにならない。

### AC-F3-04 日本の位置
- [ ] Given 推し国が日本（または日本のグループが特定可能）、When ブラケット/グループ表を見る、Then 日本の所属グループとブラケット上の位置が強調表示され、初心者が「自分の山」を識別できる。

### AC-F3-05 国への遷移
- [ ] Given グループ表/ブラケットの国名、When タップ、Then その国の国ページに遷移する。

## F4. 推し国診断

### AC-F4-01 診断フロー
- [ ] Given /quiz を開く、When 開始、Then 5〜7 問が 1 問ずつ提示され、進捗（例: 3/6）が分かる。
- [ ] Given 各設問、When 選択肢をタップ、Then 次の設問に進み、最終問の回答後に結果画面へ遷移する。

### AC-F4-02 結果表示
- [ ] Given 全問回答完了、When 結果画面、Then おすすめ 1 国 + 候補 2 国 が、各国名・国旗・おすすめ理由（1〜2 行）とともに表示される。
- [ ] Given 結果画面、When 「この国を推し国にする」をタップ、Then 推し国が localStorage に保存され、国ページまたはホームに遷移して反映される。

### AC-F4-03 やり直し
- [ ] Given 結果画面、When 「もう一度診断する」をタップ、Then 回答状態がリセットされ最初の設問から再開できる。

### AC-F4-04 診断ロジックの妥当性
- [ ] Given 同一の回答セット、When 診断を実行、Then 毎回同じ結果（決定論的）になる。
- [ ] Given 任意の回答の組み合わせ、When 結果を算出、Then 必ず 1 国以上が結果として返り、「該当なし」や空結果にならない。

## F5. 国図鑑 / 推し国ページ

### AC-F5-01 国一覧
- [ ] Given /countries を開く、When 一覧を見る、Then 出場 48 国が国名・国旗とともに表示される。
- [ ] Given 一覧、When フィルタ（グループ別 or 地域別）を操作、Then 該当国のみに絞り込まれる。

### AC-F5-02 国ページ プロフィール
- [ ] Given /countries/[code] を開く、When ページを見る、Then 国名・国旗・代表カラー・大陸/地域・強さの目安（初心者向け表現）が表示される。
- [ ] Given 代表カラー表示、When 確認、Then 色だけでなくテキストでも情報が補完される（A11y: 色依存禁止）。

### AC-F5-03 語れるネタ
- [ ] Given 国ページ、When スクロール、Then 「見どころ」「豆知識」「推し選手」がそれぞれ表示される（各機能に最低 1 件以上のデータがある）。

### AC-F5-04 新規初出場国の明示
- [ ] Given 新規初出場国（カーボベルデ / キュラソー / ヨルダン / ウズベキスタン）の国ページ、When 開く、Then 「W杯 初出場」が分かる表示がある。

### AC-F5-05 推し国保存
- [ ] Given 国ページ、When 「推し国にする」をタップ、Then localStorage に保存され、ホームの推し国カードに反映される。
- [ ] Given 既に推し国が設定済み、When 別の国を推し国にする、Then 推し国が新しい国に上書きされ、重複保存されない。

### AC-F5-06 国 → 試合
- [ ] Given 国ページ、When その国の試合一覧を見る、Then その国が関わる試合（確定分）が日本時間で表示される。

## F7. 用語じてん（用語集 + 検索 + ポップオーバー）

### AC-F7-01 用語一覧
- [ ] Given /glossary を開く、When 一覧を見る、Then 30 語以上の用語が見出し + 短い語義で一覧表示される。

### AC-F7-02 検索
- [ ] Given /glossary の検索欄、When 用語名の一部を入力、Then 一致する用語に絞り込まれる。
- [ ] Given 一致する用語がない検索語、When 入力、Then 「該当する用語がありません」と表示され、空白画面にならない。

### AC-F7-03 本文ポップオーバー
- [ ] Given 任意のページ本文中の `<Term>`（専門用語）、When タップ、Then その用語の語義がポップオーバー/ツールチップで表示される。
- [ ] Given ポップオーバー表示中、When 外側をタップ or 閉じる操作、Then ポップオーバーが閉じる。
- [ ] Given ポップオーバー、When 「詳しく」等のリンクをタップ（ある場合）、Then 用語じてんの該当項目に遷移する。

### AC-F7-04 用語網羅（UX 原則の担保）
- [ ] Given P0 全ページの本文、When レビューする、Then 初心者が知らない可能性の高い専門用語（例: オフサイド、PK、グループステージ、ラウンド32、ノックアウト等）が `<Term>` 化されている（未対応の生用語があれば不具合）。

## エッジケース / エラー状態

### AC-E-01 試合データがない日
- [ ] Given 当日に試合が 0 件（休養日・開幕前・閉幕後）、When ホーム / 今日タブを開く、Then 「今日は試合がありません」+ 次（または前）の試合日への導線が表示され、空配列でクラッシュしない。

### AC-E-02 診断の途中離脱・再訪
- [ ] Given 診断を途中まで回答、When ページを離脱して再訪、Then 中断状態の復元有無に関わらずエラーにならず、少なくとも最初から再開できる（復元する場合は復元、しない場合はリセット表示）。
- [ ] Given 診断を結果まで完了済み、When /quiz を再訪、Then 再診断 or 前回結果のいずれかが破綻なく提示される。

### AC-E-03 localStorage 無効環境
- [ ] Given ブラウザの localStorage が無効 / 容量超過 / プライベートモードで書込不可、When 推し国保存・診断結果保存・予想保存を試みる、Then アプリがクラッシュせず、セッション内では機能が動作する（または「この環境では保存できません」を穏やかに通知）。
- [ ] Given localStorage 無効環境、When ホーム・国図鑑・用語・ブラケットの閲覧機能を使う、Then 保存に依存しない閲覧系は通常どおり利用できる。

### AC-E-04 推し国の重複・変更・不正値
- [ ] Given 推し国を設定済み、When 同じ国を再度「推し国にする」、Then 重複や二重表示が起きず冪等に処理される。
- [ ] Given localStorage に存在しない / 不正な国コードが保存されている（手動改変・旧データ）、When ホームを開く、Then クラッシュせず「推し国未設定」相当にフォールバックする。

### AC-E-05 ノックアウト未確定（TBD）表示の一貫性
- [ ] Given 対戦カードが未確定のノックアウト枠、When 試合一覧 / 試合詳細 / ブラケット のいずれを見ても、Then 一貫した TBD 表記（空欄・undefined・null・記号のみが出ない）になる。

### AC-E-06 オフライン閲覧
- [ ] Given 一度オンラインで読み込み済み、When ネット接続を切って静的ページ（ホーム・国図鑑・用語・ブラケットの静的部分）を再表示、Then 静的コンテンツが閲覧できる（致命的な白画面にならない）。

### AC-E-07 存在しない ID / ルート
- [ ] Given 存在しない試合 ID（/matches/9999）または国コード（/countries/xxx）、When アクセス、Then 404 相当の親切な画面（一覧へ戻る導線つき）が表示され、クラッシュしない。

### AC-E-08 時刻・日付の境界
- [ ] Given 日本時間で日付を跨ぐ深夜キックオフ試合、When 一覧の日付グルーピングを見る、Then 日本時間基準の正しい日付に分類される。
- [ ] Given 「今日」判定、When 端末時刻が日付の境界（0:00 直後 / 23:59）、Then 日本時間基準で当日試合が正しく抽出される。

## （P1 先行記載）F6 ルール図鑑 / F8 勝敗予想 受入条件メモ

P1 着手時の参照用に方針のみ記載（P0 では非対象）。

### F6 ルール図鑑（P1）
- [ ] Given /rules のオフサイド体験、When ライン/選手を操作、Then オフサイド/オンサイドの判定が即時にフィードバックされる。
- [ ] Given ポジション・カードのセクション、When 閲覧、Then 短いステップで分割され、各セクションに `<Term>` が付く。

### F8 勝敗予想（P1）
- [ ] Given グループステージの試合、When 勝敗（または引分）を選んで保存、Then localStorage に予想が保持され、再訪時に復元される。
- [ ] Given 同一試合に再度予想、When 別の結果を選ぶ、Then 予想が上書きされ重複しない（冪等）。
- [ ] Given 結果データが整備された場合（手動入力含む）、When 予想と結果を突合、Then 的中率が計算・表示される。
- [ ] Given 結果未確定の試合、When 予想一覧を見る、Then 的中/不的中ではなく「結果待ち」と表示される。
